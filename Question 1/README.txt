Development Environment
-----------------------

To compile and run this program, you need a C compiler and the terminal. You can use any C compiler that supports the C99 standard, such as GCC You also need a text editor or an IDE to edit the source code.

My Development Environment:

Operating System: Windows 11
Compiler: GCC 10.2.0
Integrated Development Environment: Visual Studio 

------------------------------------------------------------------------------------------------------------------------------------------

To Compile the Source Code:
---------------------------

follow the steps below:
- Open the terminal
- Navigate to the directory where the source code is located.
- Type the following command to compile the source code:      " gcc Source.c -o app.exe "
- Run the resulted app.exe program (Either by double clicking on it or typing in terminal "app.exe")
